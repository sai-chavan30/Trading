# Trading
Automated Stock Momentum Trading System Python, Flask, Selenium, Socket.IO, Robinhood API

Developed a real-time automated trading system that streams Top Movers from the Robinhood API, applies a momentum-based buy/sell strategy, and executes trades automatically on a simulated trading platform using Selenium. Built a live dashboard displaying real-time price movements, open positions, and profit/loss metrics with persistent trade logging and performance tracking.
